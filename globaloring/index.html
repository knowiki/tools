O-Ring Size Standards / O型圈尺寸标准

An o-ring is one of the most common types of seals in the world. O-Rings are used in a variety of applications in different industries around the world. Because of its popularity, there is virtually an unlimited number of possible sizes of an o-ring. Thus, the need for size standards which cover the most common sizes used.
O形圈是世界上最常见的密封类型之一。 O形圈在全球不同行业中用于各种应用。由于其受欢迎程度，O形圈实际上有无限多种可能的尺寸。因此，需要覆盖最常用尺寸的尺寸标准。

Over the years, many size standards have been developed for o-rings. The table below shows what we believe to be the most often used/referenced standards by our customers in the United States and other countries.
多年来，已经为O形圈开发了许多尺寸标准。下表显示了我们认为是美国和其他国家/地区的客户最经常使用/参考的标准。

Therefore, we recommend referral to the actual specifications before proceeding with design work, etc.**
**此信息仅用作从各种国际标准中识别O型圈尺寸的指南。尽管我们认为此信息是正确的，但我们不能保证每个标准的准确性均为100％。因此，建议在进行设计工作等之前参考实际规格**。

AS568

AS568 is the most commonly used standard in the United States for defining “inch sizes” of o-rings.
AS568是美国最常用的用于定义O形圈“英寸尺寸”的标准。

BS 1806

British Standards Institution and lists all Imperial standard sizes. It has been superseded by BS ISO 3601.
英国标准协会，并列出所有英制标准尺寸。它已被BS ISO 3601取代。

IS0 6149

ISO 6149 specifies o-rings designed for use in Metric Tube Fittings. Includes 13 different sizes and calls out the metric thread size for each o-ring size.
ISO 6149规定了用于公制管件的O形圈。包括13种不同的尺寸，并为每种O形圈尺寸标注公制螺纹尺寸。

Tolerances / 公差:

Some of the standards referenced here specify the acceptable tolerance ranges for the o-rings within that standard. However, With the exception of AS568 Size Chart, other size reference charts do not list tolerances, as these tend to vary between the standards issuing organizations. To view the size tolerances for the o-rings we sell, please refer to either the AS568 Size Chart for American Standard Sizes, or Metric O-Ring Tolerances page for all other “metric” o-rings.
此处引用的某些标准规定了该标准内O形圈的可接受公差范围。但是，除了AS568尺寸表外，其他尺寸参考表没有列出公差，因为这些公差在标准发布组织之间往往会有所不同。要查看O形圈的尺寸公差，请参考美国标准尺寸的 AS568 尺寸表，或参阅所有其他“公制” O形圈的“公制”O形圈公差页面。
