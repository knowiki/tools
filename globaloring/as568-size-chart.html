AS568 O-Ring Size Chart / AS568 O型圈尺寸表

The Global O-Ring and Seal Size Reference Chart covers the AS568 O-Ring Sizes published by the Society of Automotive Engineers (S.A.E.), and includes the Extended Sizes in the 400 Series. Dimensional tolerances for all AS568 o-rings offered by Global O-Ring and Seal are in accordance with ISO 3601-1A. The purpose of this chart is to specify the inside diameters, cross-sections, tolerances, and size identification codes (dash numbers) for o-rings used in sealing applications and for straight thread tube fitting boss gaskets.

History / 历史

This size chart was first published in 1971 by the A-6c2 Seals Committee at SAE (Society of Automotive Engineers) as Aerospace Standard AS568. These standards were later republished in 1974 as AS568A, in 2001 as AS568B, in 2008 as AS568C, in 2014 as AS568D, in 2020.03 as AS568E, and most recently in 2020.09 as AS568F. Any of these terms (AS568, AS568A, AS568B, AS568C, AS568D, AS568F) may be used interchangeably when referring to an o-ring dash size.
该尺寸表于1971年由SAE（汽车工程师学会）的A-6c2密封委员会作为航空航天标准AS568发布。这些标准后来在1974年重新发布为AS568A，在2001年重新发布为AS568B，在2008年重新发布为AS568C，在2014年重新发布为AS568D，在2020年3月重新发布为AS568E，最近在2020年9月重新发布为AS568F。当提及O型圈划线尺寸时，这些术语（AS568，AS568A，AS568B，AS568C，AS568D，AS568E，AS568F）中的任何一个都可以互换使用。

Actual vs. Nominal / 实际与公称

To assist our customers in selecting the correct size o-ring we have included 3 measurements for cross reference purposes; the “Nominal”, “Actual Inch”, and “Actual Metric” sizes. We are often asked, “What is the difference between Actual and Nominal sizes for o-rings?”. The Nominal (fractional) dimensions are in name only (often used when verbally describing the size). For more accurate measurements, please refer to the actual inch or millimeter sizes, as these are the true dimensions of the o-ring.
为了帮助我们的客户选择正确尺寸的O形圈，我们提供了3种测量值供参考。 “公称”，“实际英寸”和“实际公制”尺寸。我们经常被问到：“ O形圈的实际尺寸和标称尺寸有什么区别？”。公称（分数）尺寸仅是名称（在口头描述尺寸时经常使用）。为了获得更准确的测量结果，请参考实际的英寸或毫米尺寸，因为这是O形圈的真实尺寸。

O-Ring Formulas / O型圈公式:

To calculate an o-ring’s outside diameter (OD) you may use this formula: 2(CS) + ID = OD
要计算O形圈的外径（OD），可以使用以下公式：2(CS) + ID = OD
Conversely, if you have the OD and need to find the ID use this formula: OD – 2(CS) = ID
相反，如果您有OD，并且需要查找ID，请使用以下公式：OD – 2(CS) = ID
